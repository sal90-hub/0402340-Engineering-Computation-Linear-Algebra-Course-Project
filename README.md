# Automated KVL Mesh-Current Simulator
The primary objective of this project is the development of an automated KVL mesh-current simulator: matrix-based mathematical implementation, by utilizing the MATLAB programming language to build the solver. This project aims to translate the theoretical mechanics of Kirchhoff's Voltage Law and matrix algebra into a functional software architecture. 💻

In addition to solving circuits, the goal is to show how separate linear equations come together to create a structured matrix framework to solve complex electrical networks. 🖱️
